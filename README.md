## Get started
To install, you must have **Git** installed already.

```
$ git clone https://github.com/jimmy18dev/LINE-DEVCONF-2022-tailwind-workshop.git
$ cd LINE-DEVCONF-2022-tailwind-workshop
$ npm i && npm run dev
[Open] http://localhost:3030
```
